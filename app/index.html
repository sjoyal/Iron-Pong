Iron Pong is a table tennis results tracker built for members of the 2015 Summer Front End Engineering course at The Iron Yard Orlando. Cohort members can submit scores using the submit page after logging in. Non-members are welcome to login via GitHub and comment on match results.

♥ This site is fueled by RedBull, Tanqueray's, and some AngularJS. Enjoy!
